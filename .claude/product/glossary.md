# Glossary

Domain terms used across the codebase, kept consistent between code, docs, and conversations with the business.

| Term                 | Definition                                                            |
| -------------------- | --------------------------------------------------------------------- |
| _Example: Tenant_    | _An organization account that owns its own users, data, and billing._ |
| _Example: Workspace_ | _A container for projects belonging to a single tenant._              |

Update this table whenever a new domain concept is introduced, and use these exact terms in code (variable names, entity names) rather than inventing synonyms.
